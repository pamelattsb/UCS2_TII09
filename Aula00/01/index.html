HTML

HTML (linguagem de Marcação de Hipertxto) é o bloco de construção mais basico da web. Define o significado e a assinatuira de conteúdo web.

Tag ou Marcação

O HTML usa "Marcação" para anotar texto, imagem e outros conteúdos para exibição em um navegador web.

Um elemento HTML é separdo de outro texto em um document por "tags" As tags consistem no nome de elemnto entr um "<" e ">".
