Sítense das Propriedades CSS

1. color

Define a cor do texto de um elemento

h1 {
    color: blue;    
}

2. text-aling

Alinha o texto dentro de um elemento

  • left : alinha o texto á esquerda (padrão).
  • center: centraliza o texto.
  • right: alinha o texto á direita
h1 {
    text-aling: center;    
}

3. font-size

Define o tamanho da fonte de um elemento

p {
    font-size: 18px;            
}

4. background-color

Define a cor de fundo de um elemento

button {
        background-color: yellow;
}

5. color (para botões)

Define a cor do texto de um elemento(usado em conjunto com background-color).

button{
        color:black;
}
