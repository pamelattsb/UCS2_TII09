Introdução à Tag < img >

A tag <img> em HTML é usada para incorporar imagens em uma página web. Ela é um elemento vazio, ou seja não possui uma tag de fechamento e contém apenas atributos que definem o centeúdo visual.

Atributos Principais:

  • src (source): define o caminho da imagem que será exibida. pode ser uma URL externa ou um caminho relativo dentro do servidor.

    Exemplo 1:

    <img src="./image.webp" style="width: 100%;">

    Resultado:

  • alt (alternative text): fornece um texto alternativo para ser exibido caso a imagem não possa ser carregada. Ele também melhora a acessibilidade para leitores de tela.

    Exemplo 2:

    <img src="./wp6630058.webp" alt="PEDRO PEDRO PEDRO!">

  • src também pode ser de um link externo:

Tipos Comuns de Imagens

Os formatos de imagens mais utilizados na web são:

  • (.jpg ou .jpeg): Formato utilizado amplamente para fotografias e imagens com muitos detalhes e cores. Possui compressão com perda de qualidade (lossy).
  • (.png): Usado para imagens com fundo transparente ou que nescessitam de alta qualidade, sem perda de detalhes. Utiliza compressão sem perda (lossless).
  • GIF (.gif): Formato ideal para imagens animadas e pequenos icones. Suporta trasparência, mas tem um número limitado de cores.
